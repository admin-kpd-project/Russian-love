# Реестр UX и заглушек (Russian-love)

Документ фиксирует, что считается **MVP-готово** vs **демо/временно**, и критерии «done».

## Критерии done (общие)

- **MVP-готово:** пользовательский сценарий проходит **без** фиктивных `alert` там, где заявлена фича; данные **с бэкенда** или явно подписанный офлайн-режим.
- **Демо:** помечено в UI; не выдаётся за прод-функциональность.
- **Не настроено:** кнопка скрыта или бэкенд отдаёт `paymentsEnabled: false` и т.п.

| Область | Статус | Критерий done |
|--------|--------|----------------|
| Голос / видео в чате (веб) | MVP | `MediaRecorder` → presign S3 → `POST` сообщения; плейбек `<audio>` / `<video>` по `mediaUrl`; длительность в БД/API при наличии |
| Совместимость «%» | Демо | Клиентская эвристика `compatibilityAI.ts`; **done** = подпись «оценка в приложении» или вынесение на бэк (отдельная задача) |
| События (Events) | Демо | Синтетика; **done** = баннер «Демо» или API |
| Платежи T-Bank | Зависит от env | `GET /api/payments/status` → `paymentsEnabled`; при `false` — нет «фейковой» оплаты как реальной |
| Меню чата (жалоба / удалить / очистить) | Заглушка | **done** = скрыто до API или реализованы эндпоинты |
| Лендинг «модерация/верификация» | Маркетинг | **done** = факт модерации или смягчение копирайта |
| Мобильное приложение (RN) | Отдельный пакет | `mobile-app/` + README, сборка APK вручную |
| Скачивание APK (лендинг) | Опционально | `VITE_APP_DOWNLOAD_URL` + блок; скрыт при пустом URL |
